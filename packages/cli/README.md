# luonvuitoi-honor-cli

Scaffolder + dev-server CLI for LUONVUITUOI-HONOR ROLL (`lvt-honor`). Provides `init` (scaffold a project), `import` (load CSV/Excel/JSON results), `seed` (fake data), and `dev` (run the Flask portal locally).

Install from the monorepo root:

```bash
pip install -e ./packages/core -e ./packages/cli
lvt-honor --help
```

See the repo-root `README.md` for the full toolkit overview.
